# EDUCABOT
O EducaBot é um tutor virtual desenvolvido como projeto para a disciplina de Sistemas Inteligentes, integrando Inteligência Artificial, Banco de Dados e uma estrutura completa de aprendizagem. Nosso objetivo é oferecer uma solução simples, acessível e eficiente para apoiar estudantes no processo de estudo, organização e acompanhamento do desempenho
